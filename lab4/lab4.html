¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Entre algunas de las semejanzas que encuentro entre ambos lenguajes veo que ambos son muy parecidos en sintaxis como C++. Investigando un poco, me di cuenta que ambos están enfocados a programación orientada a objetos a través de clases, objetos y herencia. Entre sus diferencias cabe destacar que Java se utiliza para el desarrollo de programas, mientras que JavaScript está enfocado en desarrollo Web. Además, Java es un entorno virtual, mientras que JS es Web. Finalmente Java fue creado por 'Sun Microsystems' y JS por 'Netscape'.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
  • toDateString(): Convierte la parte de la fecha de un objeto Date en texto

  • getDate(): Devuelve el día del mes (del 1 al 31)

  • getTime(): Devuelve el número de milisegundos desde la medianoche del 1 de enero de 1970 y una fecha especificada

  • getFullYear(): Devuelve el año
  • getMonth(): Devuelve el mes (del 0 al 11)
  • now(): Devuelve el número de milisegundos desde la medianoche del 1 de enero de 1970
¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
  • toString(): convierte un arreglo en una string de sus valores (separados por comas).

  • pop(): Elimina el último valor del arreglo

  • shift(): Elimina el primer elemento de arreglo y recorre los demás una posición anterior

  • unshift(): Añade un elemento al inicio del arreglo y recorre los demás una posición posterior
  • lenght(): Devuelve el tamaño del arreglo
  • reverse(): Voltea el orden de los elementos dentro del arreglo, del último al primero
¿Cómo se declara una variable con alcance local dentro de una función?

Se declara la variable dentro de la función con 'const','let','var'.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Es posible modificar las variables globales dentro de las funciones, pero también desde la página Web a través de una consola. Uno de los beneficios que veo es que múltiples funciones pueden acceder al valor de una misma variable al ser esta última 'global'
